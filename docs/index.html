経歴

幼稚園時代に富士通OASYSで遊んでいたことからコンピュータに興味を持ちだす。
中学の頃にN88 BasicとHTML、高校ではJavaScriptとPerlを修得し、CGIゲームを改造してサイト運営する。
大学ではソフトウェアのほか、アナログ回路や無線などハードウェアの理解を深める。
独立系BtoBソフトウェア会社へ就職し、パッケージカスタマイズをするプログラマとしてスタート。
パッケージ本体の開発にも参画したが、その後は品質向上のためプログラマからQCエンジニアへ転向。品質向上として社内勉強会を実施したり、当たり前品質であるパフォーマンスチューニングのスキルやツールを用いたセキュリティ脆弱性診断を修得する。
昨今Webサイトに対するセキュリティの脅威が増しており、セキュリティに関する理解を社内に浸透させるため、セキュリティエンジニアとしてのキャリアが始まった。
